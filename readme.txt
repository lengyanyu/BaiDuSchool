第一天
1.什么是html，什么是html5
  html：超文本标记语言。超文本即指页面内可以包含图片链接甚至音乐程序。通过标签告诉浏览器如何显示内容并被其解释执行。
  主要特点：简易性，可扩展性，平台无关，通用性。
  html5：新一代核心万维网标准，seo优化，增强了web语义性，本地存储特性，设备兼容特性，多媒体特性，三维图片特性
2.
  元素（标签加内容），属性（style），标签，内容的概念
3.文档类型
  即dtd。文档类型用于规定约束语言的定义和陈述，规范化标签语言，让解析器更好更便利的解析标记语言
4.meta标签
  描述一个html网页文档的属性，例如作者，日期，时间等等，跟搜索引擎优化有关系
5.web语义化
  对html中的标签进行语义化，让机器可以读懂内容，用机器可读的、被广泛认可的语义信息来描述内容，来降低机器处理 Web 内容的难度。提升浏览器解释效率，更好的被解析器，人类，爬虫解析内容，如section，article，thead，tfoot等等
6.链接 
  <a>标签，链接就是从一个网页指向一个目标的连接关系
7.dd，dl，dt表示列表
